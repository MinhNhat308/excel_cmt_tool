excel_cmt_tool — Flutter
========================

Cách chạy (máy đã cài Flutter SDK):

1) Mở terminal trong thư mục `excel_cmt_tool`.
2) Nếu chưa có thư mục `android`, `windows`, … thì tạo nền tảng:
   flutter create .
3) Cài gói: flutter pub get
4) Chạy (Windows desktop): flutter run -d windows
   Hoặc Chrome: flutter run -d chrome

Định dạng Excel: xem nút trợ giúp (?) trong ứng dụng.

Xuất .cmt (file binary):
1) Build exporter (một lần):
   dotnet build tools/cmt_exporter/FuGrade.csproj -c Release
2) flutter build windows
   (FuGrade.exe được copy cạnh excel_cmt_tool.exe nếu đã build bước 1)
3) Trong app: chọn Excel -> Xuất .cmt

File .cmt là binary .NET (serialize), giống file mẫu thầy gửi — không phải text.
